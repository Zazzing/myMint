# 项目描述

> 写这个项目的目的是为了学习vue2.0相关的知识点，加深对vue2.0的理解

### 如何运行

``` bash
# 下载项目
git clone https://github.com/Zazzing/myMint.git

# 安装依赖包
npm install

# 启动项目
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
